Functions for working with maps.

Functions for working with maps.

Maps in Zap are immutable key-value collections. Keys are atoms (interned identifiers), values are integers. Maps use a flat array representation optimized for small collections.

Examples

m = %{name: "Alice", age: 30}
Map.get(m, :name, 0)     # => atom ID for "Alice"
Map.has_key?(m, :name)   # => true
Map.size(m)              # => 2
