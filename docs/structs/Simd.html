Simd

Tuple-backed SIMD operations.

Tuple-backed SIMD operations.

Simd exposes Zap tuples as the public SIMD value shape and delegates the actual lane operations to Zig's @Vector builtins through :zig.Simd. No Zap vector type is introduced: {f32, f32, f32, f32} remains an ordinary tuple, and SIMD behavior is only available through this module.
